# Korea Electricity Bill Calculator (한국 전력 요금 계산기)

Homey 앱으로 한국전력(KEPCO) 누진제 요금 체계를 기반으로 전기 요금을 계산합니다.

## 주요 기능

### 실시간 요금 계산
- 한국전력 주택용 전기요금 누진제 계산
- 저압/고압 요금제 지원
- 계절별 요금 자동 적용 (하계 7-8월 / 동계 12-2월 / 기타)
- 검침일 기준 청구기간 자동 계산

### 지원 계약종별
주택용 외에 정액(계절별 단일요율) 종별도 지원합니다. 비주택용 종별은 **계약전력(kW)** 입력이 필요합니다.

- **주택용** (저압 / 고압) — 누진제
- **일반용(갑)Ⅰ** (저압 / 고압A·B, 선택Ⅰ·Ⅱ)
- **교육용(갑)** (저압 / 고압A·B, 선택Ⅰ·Ⅱ)
- **산업용(갑)Ⅰ** (저압 / 고압A·B, 선택Ⅰ·Ⅱ)
- **농사용(갑)**, **농사용(을)** (저압 / 고압)
- **심야전력(갑)** (월 최저요금 적용)
- **일반용(갑)Ⅱ**, **산업용(갑)Ⅱ** (고압A·B, 선택Ⅰ·Ⅱ) — **시간대별(TOU)**

> **시간대별(TOU) 요금제**는 설치 이후 실시간 사용량을 경/중/최대부하로 집계해 계산합니다. 설치 후 **첫 완전한 청구기간부터** 정확하며, 미터 보고 시점 기준 근사입니다. 경부하 22-08시, 여름·봄가을 최대부하 15-21시, 겨울 최대부하 09-12·16-19시.
>
> 계약전력 300kW 이상(을) 종별과 심야전력(을)은 대형시설/복잡한 요금구조라 아직 지원하지 않습니다.

### 할인 제도 지원 (주택용 전용)
- **대가족 할인**: 5인 이상 가구, 출산가구, 3자녀 이상, 생명유지장치
- **복지 할인**: 유공자/장애인, 사회복지시설, 기초생활수급자(생계/의료/주거/교육), 차상위계층

### Flow 자동화

#### 트리거 (When...)
- **누진 단계 변경**: 누진 단계가 변경되었을 때 (예: 1단계 → 2단계, 검침일 초기화 포함)
- **누진 단계 상승**: 누진 단계가 올라갈 때만 (검침일 초기화로 내려갈 때는 제외)

#### 조건 (And...)
- **요금 초과 확인**: 이번달 사용량 요금이 특정 금액을 초과하는지 확인

## 요금 체계

### 누진 구간 (주택용 저압)
| 구간 | 사용량 | 기본요금 | 전력량요금 |
|------|--------|----------|------------|
| 1단계 | 0~200kWh | 910원 | 120.0원/kWh |
| 2단계 | 201~400kWh | 1,600원 | 214.6원/kWh |
| 3단계 | 401kWh~ | 7,300원 | 307.3원/kWh |

*하계(7~8월): 300/450/1000kWh 구간 적용
*동계(12~2월): 1000kWh 초과 시 슈퍼유저 요금 적용

### 부가 요금
- 기후환경요금
- 연료비조정액
- 부가가치세 (10%)
- 전력산업기반기금 (2.7%, 2025.7.1 인하)

## 설치 방법

1. Homey 앱에서 "Korea Electricity Bill Calculator" 검색
2. 앱 설치
3. 디바이스 추가 시 `meter_power` capability가 있는 전력 미터 기기 선택
4. 설정에서 검침일, 계약종별(비주택용은 계약전력도), 전압 구분, 할인 옵션 설정

### 설치 후 권장 설정 (미터값 기준점)

이 앱은 소스 기기의 **누적 사용량(meter_power)** 을 읽어, 검침일·연초 대비 사용량을 계산합니다. 최초 설치 시 월초·연초 미터값이 **현재 값으로 자동 설정**되므로, 설치 시점부터 0에서 사용량이 시작됩니다.

정확한 이번달/올해 사용량을 원한다면, 디바이스 **고급 설정**에서 아래 값을 직접 입력하세요:

- **월초(검침일) 미터값**: 지난 검침일 당시 전력량계 미터값 (kWh)
- **연초 미터값**: 1월 1일 당시 전력량계 미터값 (kWh)

> 월 중간에 설치했다면 이 값을 입력해야 검침일 기준 이번달 사용량이 정확히 계산됩니다. 값을 모르면 그대로 두어도 되며, 다음 검침일/연초부터는 자동으로 정확해집니다.

## 설정 옵션

### 한전 설정
| 설정 | 설명 | 기본값 |
|------|------|--------|
| 검침일 | 한전 검침일 (1-31, 0은 말일) | 1 |
| 계약종별 | 주택용 / 일반용(갑)Ⅰ·Ⅱ / 교육용(갑) / 산업용(갑)Ⅰ·Ⅱ / 농사용(갑)·(을) / 심야전력(갑) | 주택용 |
| 전압 구분 (주택용 전용) | 저압(단독주택) / 고압(주로 아파트) — 계약종별이 '주택용'일 때만 사용 | 저압 |
| 계약전력 (kW) | 비주택용 종별의 기본요금 계산에 필요 (기본요금 = 원/kW × 계약전력). 주택용·심야전력(갑)은 0 | 0 |

### 할인 설정
| 설정 | 옵션 |
|------|------|
| 대가족 할인 | 없음 / 5인이상·출산·3자녀 / 생명유지장치 |
| 복지 할인 | 없음 / 유공자·장애인 / 사회복지시설 / 기초생활(생계·의료) / 기초생활(주거·교육) / 차상위계층 |

### 미터 값 설정
| 설정 | 설명 |
|------|------|
| 총 사용량 시작값 | 소스 기기 미터값에 더할 오프셋 (kWh) |
| 월초(검침일) 미터값 | 지난 검침일 당시 미터값 (kWh). 이번달 사용량 계산 기준점. 최초 설치 시 현재 값으로 자동 설정, 이후 검침일마다 자동 갱신 |
| 연초 미터값 | 1월 1일 당시 미터값 (kWh). 올해 누적 사용량 계산 기준점. 최초 설치 시 현재 값으로 자동 설정, 이후 연초마다 자동 갱신 |
