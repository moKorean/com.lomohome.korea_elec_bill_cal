# Korea Electricity Bill Calculator (한국 전력 요금 계산기)

Homey 앱으로 한국전력(KEPCO) 누진제 요금 체계를 기반으로 전기 요금을 계산합니다.

## 주요 기능

### 실시간 요금 계산
- 한국전력 주거용 전기요금 누진제 계산
- 저압/고압 요금제 지원
- 계절별 요금 자동 적용 (하계 7-8월 / 동계 12-2월 / 기타)
- 검침일 기준 청구기간 자동 계산

### 할인 제도 지원
- **대가족 할인**: 5인 이상 가구, 출산가구, 3자녀 이상, 생명유지장치
- **복지 할인**: 유공자/장애인, 사회복지시설, 기초생활수급자(생계/의료/주거/교육), 차상위계층

### 센서 (Capabilities)

| 센서 | 설명 |
|------|------|
| **이번달 예상 요금** | 현재 사용량 기준 이번 청구기간 예상 요금 (₩) |
| **이번달 사용량** | 검침일 기준 현재 청구기간 사용량 (kWh) |
| **누진 단계** | 현재 적용 중인 누진 단계 (1~4단계) |
| **현재 단가** | 평균 전력량 요금 단가 (₩/kWh) |
| **이번 시간 사용** | 현재 시간 사용량 (kWh) |
| **지난 시간 사용** | 직전 1시간 사용량 (kWh) |
| **오늘 사용량** | 오늘 0시부터 현재까지 사용량 (kWh) |
| **어제 사용** | 어제 하루 총 사용량 (kWh) |
| **일평균 사용량** | 이번 청구기간 일평균 사용량 (kWh) |
| **지난달 대비** | 예상 사용량과 지난달 실제 사용량 비교 (%) |
| **예상 사용량** | 월말 예상 총 사용량 (kWh) |
| **예상 요금** | 월말 예상 총 요금 (₩) |
| **지난달 사용** | 지난 청구기간 총 사용량 (kWh) |
| **지난달 요금** | 지난 청구기간 총 요금 (₩) |
| **올해 사용량** | 연초부터 현재까지 누적 사용량 (kWh) |
| **올해 누적 요금** | 연초부터 현재까지 누적 요금 (₩) |
| **총 사용량** | 소스 기기 총 누적 사용량 (kWh) |

### Flow 자동화

#### 트리거 (When...)
- **누진 단계 변경**: 누진 단계가 변경되었을 때 (예: 1단계 → 2단계)

#### 조건 (And...)
- **요금 초과 확인**: 이번달 예상 요금이 특정 금액을 초과하는지 확인

## 요금 체계

### 누진 구간 (주거용 저압)
| 구간 | 사용량 | 기본요금 | 전력량요금 |
|------|--------|----------|------------|
| 1단계 | 0~200kWh | 910원 | 120.0원/kWh |
| 2단계 | 201~400kWh | 1,600원 | 214.6원/kWh |
| 3단계 | 401kWh~ | 7,300원 | 307.3원/kWh |

*하계(7~8월): 300/450/1000kWh 구간 적용
*동계(12~2월): 1000kWh 초과 시 슈퍼유저 요금 적용

### 부가 요금
- 기후환경요금
- 연료비조정액
- 부가가치세 (10%)
- 전력산업기반기금 (3.7%)

## 설치 방법

1. Homey 앱에서 "Korea Electricity Bill Calculator" 검색
2. 앱 설치
3. 디바이스 추가 시 `meter_power` capability가 있는 전력 미터 기기 선택
4. 설정에서 검침일, 전압 구분, 할인 옵션 설정

## 설정 옵션

### 한전 설정
| 설정 | 설명 | 기본값 |
|------|------|--------|
| 검침일 | 한전 검침일 (1-31, 0은 말일) | 1 |
| 전압 구분 | 저압(일반 가정) / 고압 | 저압 |

### 할인 설정
| 설정 | 옵션 |
|------|------|
| 대가족 할인 | 없음 / 5인이상·출산·3자녀 / 생명유지장치 |
| 복지 할인 | 없음 / 유공자·장애인 / 사회복지시설 / 기초생활(생계·의료) / 기초생활(주거·교육) / 차상위계층 |

### 미터 값 설정
| 설정 | 설명 |
|------|------|
| 총 사용량 시작값 | 소스 기기 미터값에 더할 오프셋 (kWh) |
| 월초(검침일) 미터값 | 검침일 기준 미터값 (자동 업데이트) |
| 연초 미터값 | 연초 기준 미터값 (자동 업데이트) |
